# 登录日志


## 新增登录日志


**接口地址**:`/api/login-log`


**请求方式**:`POST`


**请求数据类型**:`application/x-www-form-urlencoded,application/json`


**响应数据类型**:`*/*`


**接口描述**:


**请求示例**:


```javascript
{
  "id": 0,
  "userId": 0,
  "username": "",
  "ipAddr": "",
  "browser": "",
  "loginStatus": 0,
  "msg": "",
  "createTime": ""
}
```


**请求参数**:


| 参数名称 | 参数说明 | 请求类型    | 是否必须 | 数据类型 | schema |
| -------- | -------- | ----- | -------- | -------- | ------ |
|sysLoginLog|登录日志|body|true|SysLoginLog|SysLoginLog|
|&emsp;&emsp;id|日志ID||false|integer(int64)||
|&emsp;&emsp;userId|用户ID||false|integer(int64)||
|&emsp;&emsp;username|用户名||false|string||
|&emsp;&emsp;ipAddr|登录IP||false|string||
|&emsp;&emsp;browser|浏览器信息||false|string||
|&emsp;&emsp;loginStatus|登录状态：0-失败，1-成功||false|integer(int32)||
|&emsp;&emsp;msg|状态消息||false|string||
|&emsp;&emsp;createTime|创建时间||false|string(date-time)||


**响应状态**:


| 状态码 | 说明 | schema |
| -------- | -------- | ----- | 
|200|OK|ResultObject|


**响应参数**:


| 参数名称 | 参数说明 | 类型 | schema |
| -------- | -------- | ----- |----- | 
|code|状态码|integer(int32)|integer(int32)|
|msg|消息|string||
|data|数据|object||


**响应示例**:
```javascript
{
	"code": 200,
	"msg": "success",
	"data": {}
}
```
