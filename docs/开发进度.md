## 视觉与交互设计（深邃极光）

- **主题系统**：`style.css` 用 `:root`（亮）/ `html[data-theme="dark"]`（暗）双套语义令牌（玻璃层、极光三色、品牌渐变、文字层、光晕/阴影）。组件只引用变量，切主题不改组件。工具类 `.aurora-bg`（多层光斑漂移）、`.glass-card`（毛玻璃）。
- **明暗切换**：`store/theme.js` 管理，`main.js` 挂载前 `bootstrapTheme()` 防首屏闪白，并引入 `element-plus/theme-chalk/dark/css-vars.css` 让 EP 组件跟随。切换入口在侧边栏底部与登录页右上角，状态存 localStorage。
- **写日记页**：上方「成稿区」——未生成时空态引导，生成/编辑时为标题+正文（可手动改），顶部放日期/风格元信息与字数；下方「气泡对话流」——用户素材气泡靠右、AI 回复靠左，回车发送、Shift+回车换行，媒体按钮占位。AI 生成接口接入后把内容填入成稿区（已留 `fillToDraft`）。
- **个人中心**：默认展示态（名片 Banner + 基本资料/联系方式只读分组，空字段显示「未填写」），点「编辑资料」就地展开分组表单，保存回展示态、取消还原。

## 三、已对接的后端接口

| 功能 | 方法 | 路径 |
| --- | --- | --- |
| 用户登录 | POST | `/api/auth/login` |
| 用户注册 | POST | `/api/auth/register` |
| 查询用户 | GET | `/api/user/{id}` |
| 更新用户 | PUT | `/api/user` |
| 日记分页 | GET | `/api/diary/page` |
| 日记详情 | GET | `/api/diary/{id}` |
| 新增日记 | POST | `/api/diary` |
| 更新日记 | PUT | `/api/diary` |
| 删除日记 | DELETE | `/api/diary/{id}` |
| 多媒体分页/新增/删除 | GET/POST/DELETE | `/api/media`、`/api/media/page`、`/api/media/{id}` |
